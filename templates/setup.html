SETUP GUIDE

1. REISERX DRIVER SETUP

1. Install the app in device A

2. Grant Notification access permission from Settings/Apps/Special access

3. The app will be launched automatically

4. Complete the checklist displayed in app

5. Login

6. A QR code with device code will be displayed

Done

1.1 REQUIRED PERMISSIONS FOR REISERX DRIVER

1. Call logs - for getting call logs Files and media (for Android 11+ Give all files access) - for file operations

2. Contacts - for getting contacts

3. Files and media (for Android 11+ Give all files access) - for file operations

4. Location (for android 11+ give background location permission) - to trace the device

5. Microphone - for audio recording

6. Draw over other apps/Appear on top - for perfect launching of the app Special permissions for ReiserX

7. Ignore battery optimization - for service to run permanently

8. Notification access - to record notifications and service restart

9. Usage access - to get device usage stats

10. Accessibility access for ReiserXOn launching the app you will be redirected to Accessibility settings from there you have to Start Accessibility service for ReiserX. Accessibility service is used to Record audio in all devices & Capture screenshot in android 11+ ReiserX for some devices

11. Xiaomi/Oppo/Vivo & other phones with cusom UI: you have to manually enable Background data access and Auto restart permission from App info

2. REISERX PANEL SETUP

1. Install the app in Device B

2. Create an account for you

3. Verify your account

4. After successfully login in you will be moved to Home page of the app

5. From there tap on + icon button

6. Enter the ReiserX driver device code or SCAN the QR code displayed on other device in which ReiserX driver is installed

You're all set to go, now you can remote access device A and fetch data
